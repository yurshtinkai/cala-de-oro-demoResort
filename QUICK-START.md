## What Each Role Can Do

### 👨‍💼 MANAGER (Manager Role)
```
✅ Input all transactions
✅ Edit any transaction
✅ Delete any transaction
✅ View financial reports
✅ Full operational control
✅ Review IT Staff submissions
```

**Login → See Everything + Edit/Delete Buttons**

---

### 💻 IT STAFF (Data Entry)
```
✅ Input all transactions
✅ Create bookings
✅ View room status
❌ Cannot edit after submit
❌ Cannot delete records
❌ Cannot view reports
```

**Login → Input Data Only (No Edit/Delete Buttons)**

---

### 👔 CEO (Executive)
```
✅ View all reports
✅ Executive dashboard
✅ Analytics & insights
❌ Cannot input data
❌ Cannot edit anything
❌ Read-only access
```

**Login → Auto-opens Reports Dashboard**

---

## Quick Actions

### Add Transaction (IT Staff or Manager)
1. Login
2. Go to any transaction section
3. Enter amount and click "Add to Bill"
4. Transaction appears with your name

### Edit Transaction (Manager Only)
1. Find transaction in cart
2. Click "✏️ Edit"
3. Enter new amount
4. Click OK
5. Totals auto-update

### Delete Transaction (Manager Only)
1. Find transaction in cart
2. Click "🗑️ Delete"
3. Confirm deletion
4. Transaction removed

### View Reports (Manager or CEO)
1. Click "Reports & Analytics"
2. View all data
3. Print or export as needed

## Common Workflows

### Scenario 1: Customer Check-in
```
IT Staff Action:
1. Login as IT Staff
2. Go to "Room Status Map"
3. Click available room
4. Enter guest details
5. Confirm booking
→ Transaction recorded with IT Staff name
```

### Scenario 2: Correcting an Error
```
Manager Action:
1. Login as Manager
2. Review transactions
3. Find incorrect amount
4. Click "Edit"
5. Change to correct amount
→ System recalculates totals
```

### Scenario 3: Daily Report Review
```
CEO Action:
1. Login as CEO
2. Auto-opens Reports
3. Review KPIs
4. Check occupancy rate
5. Export for meeting
→ Read-only, no modifications
```

## Troubleshooting

### "Access Denied" Message?
→ You're trying to access a restricted feature  
→ Check your role permissions above

### Can't Login?
→ Verify you're using correct email  
→ Remember: admin@ is deprecated, use manager@

### Don't See Edit Buttons?
→ Only MANAGER role has edit/delete  
→ IT Staff cannot see these buttons

### Totals Not Updating?
→ Refresh the page  
→ Clear browser cache  
→ Check browser console for errors
